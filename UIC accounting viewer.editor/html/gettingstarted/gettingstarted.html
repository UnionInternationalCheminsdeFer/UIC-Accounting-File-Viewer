Application

This application provides to function to view and edit accounting data according to the new format in UIC IRS 300301.

The following features are implemented:

  • Import company code list as reference data.
  • Import of accounting files in the json format.
  • Creation of new accounting files in the json format to create example and test data.
  • Editing the accounting file.
  • Exporting the accounting file in json format.
  • Validating the accounting file.
  • Displaying a summary of the accounting file amounts.


The application is made of two main windows, one showing the complete data set as a tree (resource view) and one showing the content of leaves of the tree (property view). Additional views are included to display errors. Editing support is available in the property view for the content and in the context menu of the tree view for editing the tree structure.

Functions for import, conversion and export are available in the tool bar.

Intermediary results can be stored at any time.

Data structure

    The data set includes:

  • reference data code lists
  • the accounting file data
  • workflow history
  • fare data as reference data - not yet implemented

  • code lists
    • carriers - UIC company codes
